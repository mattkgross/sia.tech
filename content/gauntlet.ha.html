Intro to Sia

Similar to Amazon S3, or Google Cloud Storage, Sia is a cloud storage layer. Just like Amazon and Google, we provide an API that allows developers to upload and download files to and from the network. But unlike Amazon and Google, Sia is decentralized. Files are encrypted and spread across dozens of nodes on the network, such that there is no single point of failure. Sia is also significantly cheaper than Amazon and Google (around 10x cheaper right now). Sia uses a cryptocurrency similar to Bitcoin for all storage payments. Getting started

First you'll want to get the latest release of the Sia Daemon from our downloads page.

Syncing with Sia's blockchain for the first time can take up to a day on a slow connection. To help you get started as soon as possible, we're providing a pre-downloaded blockchain that has 99% of the blocks (up to block height 78651). You can check the current height of the blockchain at explore.sia.tech

  • 1. Download the blockchain: sia.tech/sia-data.tar.gz
  • 2. Extract it: 
    tar -xzvf sia-data.tar.gz
  • 3. Start the Sia Daemon: 
    siad -d sia-data
  • 4. Check the status of the blockchain download with 
    siac

Now that you have a Sia Daemon up and running, check out our API Quickstart Guide. More detailed descriptions of the API endpoints are available in the API Documentation.
